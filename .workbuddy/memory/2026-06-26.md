## AES-256 加密导出/解密导入优化

- **去掉 Base64**：改回零开销二进制格式
- 新格式：`QUOX01(6) + Salt(16) + IV(16) + GZip(AES(JSON))`，File.WriteAllBytes 直接写
- 兼容旧格式：automagically 检测（明文 JSON = `[`开头；旧 Base64 .quox；新二进制 .quox）
- **GZip 压缩**：AES 前先 `GZipStream(CompressionLevel.SmallestSize)`，20MB JSON → ~3MB
- 导入时解压验证失败 → `InvalidOperationException` → 统一提示「密码错误，或备份文件已损坏」
- 改 `Rfc2898DeriveBytes` → 新版 `Pbkdf2` 静态方法（消 SYSLIB0060）
